# 78V-comfyui

**ComfyUI custom nodes and workflows** for AI image generation.

## Overview

A collection of custom ComfyUI nodes and example workflows developed for 78 Ventures media production. Includes LoRAGen (LoRA training pipeline), SceneSetter (scene composition), and bear_nodes (utility nodes).

## Contents

- `nodes/` — Custom ComfyUI node definitions
- `example_workflows/` — Ready-to-use workflow JSON files
- `web/` — Web UI extensions
- `dev/` — Development and testing tools
